Hex project Home page

Hex is a simple command line tool allowing display hexadecimal representation of any input source (number, string, standard input, file).

Installation

From the folder where you unzip project, type the following commands:

make
make install

Display hexadecimal value of

A decimal (base-10) number

hex -d <number>

A character string

hex -s <string>
hex -s "<multiple> <words> <string>"
hex <string>
hex "<multiple> <words> <string>"

The standard input stream

echo <string> | hex
hex
# then type anything terminated by Ctrl+D

A file

hex -f <file_path>
cat <anyfile> | hex

Other use cases

Read a large content

hex -f <big_file> | more

Find an hexadecimal value

hex -f <file> | grep "6C 3F"

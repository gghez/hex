Hex

Display hexadecimal view of any input

Download this project as a .zip file Download this project as a tar.gz file

Welcome to Hex project Home page

Hex is a simple command line tool allowing display hexadecimal representation of any input source (number, string, standard input, file).

Usage samples:

hex -d 134
hex "john doe"
hex -s "john doe"
hex HEX

Installation

From the folder where you unzip project, type the following commands:

make
make install
